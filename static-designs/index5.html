FAQ

How does this tool work? +

NaturalTTS uses Piper, a neural speech synthesis system that converts text into speech. Everything runs directly in your browser using WebAssembly technology. The system processes your text, analyzes linguistic features, and generates natural-sounding speech that can be played back or downloaded as an MP3 file.

Is there a text limit? +

Yes, you can convert up to 5,000 characters at once (750-1000 words). This limit helps ensure optimal performance in the browser environment. For longer texts, break them into smaller chunks and process them sequentially.

What file format is the audio downloaded in? +

All generated audio is available in MP3 format (128kbps), which offers excellent audio quality with small file sizes. MP3 files are compatible with virtually all devices and media players.

Is my data private? +

Yes, completely. All processing happens locally in your browser, and no data is sent to any server. Your text never leaves your device, ensuring 100% privacy and security. This makes NaturalTTS an excellent choice for sensitive content.

Can I use the generated audio for commercial purposes? +

Yes, you can use the generated audio for personal and commercial projects. Since the processing is done locally on your device, you own the output you create. This makes NaturalTTS perfect for creating content for videos, podcasts, e-learning materials, and other media projects.

Command Reference

For technical users who prefer the command line, NaturalTTS offers a JavaScript API that can be used in the browser console.

// Initialize the TTS engine
const
tts = await NaturalTTS.init();
// Generate speech from text
const
audio = await tts.speak({   text: "Hello world, this is a text to speech example.",   voice: "en_US_female",   speed: 1.0,   format: "mp3" });
// Play the generated audio
audio.play();
// Download the audio file
audio.download("speech.mp3");
